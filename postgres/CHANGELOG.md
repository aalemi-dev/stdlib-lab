# Changelog

## [1.0.0](https://github.com/aalemi-dev/stdlib-lab/releases/tag/postgres/v1.0.0) - Initial Release

### Features

* PostgreSQL client with GORM integration
* Query builder with RETURNING clause, locking clauses, observer hooks, and row scanner
* Schema and data migrations with history tracking
* fx lifecycle support for dependency injection
* Comprehensive error translation and categorization
